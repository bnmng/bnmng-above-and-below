== Description ==
This plugin is used to add text above and below the content of post content.  You can add different text to different selections of posts, which can be selected based on post type, author,  or taxonomies such as category.  

This plugin does not alter the post in the database.

== Frequently Asked Questions ==
Will this work with custom post types?

You can apply this plugin to custom post types but there is no guarantee that this plugin will work with your post type.  Generally, the more like a regular post the custom post type is, the more likely that this plugin will work as expected with the custom post type.
